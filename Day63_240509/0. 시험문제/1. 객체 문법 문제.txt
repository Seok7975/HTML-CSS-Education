1. pet 이라는 객체를 만들자.
pet 이라는 객체에는 name, eat 속성이 있다.
name 속성에는 '구름'이라고 주자.
eat 속성에는 메소드를 주자.
pet.eat('밥'); 이라고 호출하면
"구름은/는 밥을/를 먹습니다." 라고 alert 창을 띄우자.

2. student라는 빈 객체를 만들자.
동적으로 student 객체에 이름을 '윤인성', 취미를 '악기', 장래희망을 '생명공학자'를 추가하자.
콘솔창에 다음과 같이 JSON 형태로 출력하자.

{
  "이름": "윤인성",
  "취미": "악기",
  "장래희망": "생명공학자"
}

3. 2번에서 만든 student 객체에서 동적으로 student의 장래희망을 삭제하자.
그리고 콘솔창에 다음과 같이 JSON 형태로 출력하자.

{
  "이름": "윤인성",
  "취미": "악기"
}

4. 1번에서 만든 pet 객체의 eat 메소드를 메소드 간단 선언 구문으로 바꿔보자.
pet.eat('밥'); 이라고 호출해보자.

5. test라는 객체를 만들고 a라는 속성에 익명함수를 주자. 그리고 콘솔창에서 this를 출력해보자. b라는 속성을 만들고 속성에 화살표함수를 주자. 그리고 콘솔창에서 this를 출력해보자.
