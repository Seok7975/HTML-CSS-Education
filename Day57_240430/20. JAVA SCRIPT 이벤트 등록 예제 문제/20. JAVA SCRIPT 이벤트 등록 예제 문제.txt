menu.html를 만들자.
hello.html를 만들자.

menu.html에 ClickMe 라는 버튼을 만들자.
ClickMe 라는 버튼을 누르면 hello.html로 이동하게끔하자.
(클릭 이벤트가 발생하게 프로그래밍한다.)

hello.html에서는 back이라는 버튼을 누르면 전 페이지로 이동하게 하자.

이벤트 리스너(event listener)
이벤트 리스너란 이벤트가 발생했을 때 그 처리를 담당하는 함수를 가리키며, 이벤트 핸들러(event handler)라고도 합니다.
지정된 타입의 이벤트가 특정 요소에서 발생하면, 웹 브라우저는 그 요소에 등록된 이벤트 리스너를 실행시킵니다.
